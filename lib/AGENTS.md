# lib Context

## Purpose:
The `lib` directory serves as the root orchestration layer for the MangaStash application. It is responsible for the application's entry point, global dependency injection initialization, and the high-level routing configuration that integrates various modular features into a cohesive user experience.

## Key Components:
* **main.dart**: The application entry point; initializes the `ServiceLocator`, registers core and domain registrars, and handles global error reporting.
* **main_route.dart**: Defines the primary navigation structure using `GoRouter`, aggregating routes from feature modules and setting up the `StatefulShellRoute` for the main application shell.
* **main_path.dart**: Centralizes string constants for top-level navigation paths to ensure consistency across the app.
* **firebase_options.dart**: Provides platform-specific Firebase configuration generated by the FlutterFire CLI.
* **screen/**: Contains foundational application screens including the `SplashScreen` for initialization, `AppsScreen` for lifecycle management, and `MainScreen` which acts as the navigation shell.

## Dependencies:
* **Core Modules**: `core_route`, `core_network`, `core_storage`, `core_analytics`, `core_environment`.
* **Feature Modules**: `feature_library`, `feature_updates`, `feature_history`, `feature_browse`, `feature_more`, `feature_common`.
* **Domain Modules**: `domain_manga`.
* **UI/Library**: `ui_common`, `service_locator`.
* **External**: `flutter`, `go_router`, `firebase_core`.

## Local Conventions:
* **Modular Initialization**: Each domain or core module provides a `Registrar` class, which is called in `main.dart` to populate the `ServiceLocator`.
* **Route Aggregation**: `MainRouteBuilder` uses an aggregation pattern, collecting route definitions from individual feature modules rather than defining them all in-place.
* **Wrapper Pattern**: The root of the app uses a `WrapperScreen` to manage the transition from a loading (splash) state to the main application or an error state.
* **Shell Navigation**: Implements `StatefulShellRoute.indexedStack` to maintain state across different bottom navigation branches.
